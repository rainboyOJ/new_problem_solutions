## 题目解析

这是一个非常经典的**二分图最大权完美匹配**问题。

### 1. 题目含义与计算示例

题目含义：

题目要求我们把 $n$ 个男运动员和 $n$ 个女运动员一一配对。

每一对（男 $i$, 女 $j$）的“混合双打优势”计算公式为：



$$\text{单组优势} = P_{i,j} \times Q_{j,i}$$

- $P_{i,j}$：男 $i$ 对女 $j$ 的配合度（查矩阵 $P$ 的第 $i$ 行第 $j$ 列）。
- $Q_{j,i}$：女 $j$ 对男 $i$ 的配合度（查矩阵 $Q$ 的第 $j$ 行第 $i$ 列）。

我们需要找到一种配对方案（每人只能配一次），使得所有配对的**优势之和**最大。

**样例极速解析：**

输入数据：

$n=3$

$P$ 矩阵 (男对女):

```
10 2 3  (男1对: 女1=10, 女2=2, 女3=3)
2 3 4   (男2对: 女1=2,  女2=3, 女3=4)
3 4 5   (男3对: 女1=3,  女2=4, 女3=5)
```

$Q$ 矩阵 (女对男):

```
2 2 2   (女1对: 男1=2, 男2=2, 男3=2)
3 5 3   (女2对: 男1=3, 男2=5, 男3=3)
4 5 1   (女3对: 男1=4, 男2=5, 男3=1)
```

**最优配对方案（题目输出 52 的来源）：**

1. **男 1 + 女 1**：
   - $P_{1,1} = 10$
   - $Q_{1,1} = 2$
   - 优势 = $10 \times 2 = 20$
2. **男 2 + 女 3**：
   - $P_{2,3} = 4$
   - $Q_{3,2} = 5$ （注意是 $Q$ 的第 3 行第 2 列）
   - 优势 = $4 \times 5 = 20$
3. **男 3 + 女 2**：
   - $P_{3,2} = 4$
   - $Q_{2,3} = 3$ （注意是 $Q$ 的第 2 行第 3 列）
   - 优势 = $4 \times 3 = 12$

**总和** = $20 + 20 + 12 = 52$。

------

### 2. 建模思路：最大费用最大流

这道题和之前的“分配问题”本质上一模一样，都是**二分图最佳匹配**。

**网络流模型构建：**

1. **节点设置**：
   - 源点 $S$ ($0$)。
   - 男运动员节点 $1 \dots n$。
   - 女运动员节点 $n+1 \dots 2n$。
   - 汇点 $T$ ($2n+1$)。
2. **边的建立**：
   - **源点 $\to$ 男运动员**：容量 $1$，费用 $0$（限制每个男生只能选一个搭档）。
   - **女运动员 $\to$ 汇点**：容量 $1$，费用 $0$（限制每个女生只能选一个搭档）。
   - **男 $i \to$ 女 $j$**：
     - 容量：$1$。
     - **费用**：$-(P_{i,j} \times Q_{j,i})$。
     - **原理**：因为我们要**最大化**总优势，而费用流算法计算的是**最小**费用，所以我们将收益取负值。最后取结果的相反数即可。
3. **求解**：
   - 跑最小费用最大流 (MCMF)。
   - 答案为 `-mcmf.min_cost`。
